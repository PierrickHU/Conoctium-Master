## -------------- Contrôles ---------------
Les contrôles du jeu sont désignés par rapport à une manette de XBOX360.
Le jeu se joue à deux joueurs.

- Joystick: Mouvement de la boule
- A: Saut
- B: Annuler
- RT: Se repousser de l'autre joueur
- LT: S'attirer vers l'autre joueur
